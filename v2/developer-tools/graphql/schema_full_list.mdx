## Object

### CustodialWallet

Represents a custodial wallet managed by Cobo.

#### Implements

`CustodialWallet` implements the following interfaces:

- `Wallet`
- `Node`

#### Fields

| Field Name              | Type                     | Description                                      |
| ----------------------- | ------------------------ | ------------------------------------------------ |
| `id`                    | `ID`                    | Unique identifier of the wallet.                 |
| `walletId`              | `String`                | Wallet ID assigned by the platform.              |
| `walletType`            | `WalletType`            | Type of the wallet (e.g., HOT, COLD).            |
| `walletSubtype`         | `WalletSubtype`         | Subtype of the wallet (e.g., MPC).               |
| `name`                  | `String`                 | Wallet name.                                     |
| `addresses`             | `AddressConnection`      | List of addresses associated with this wallet.   |
| `transactions`          | `TransactionConnection`  | List of transactions related to this wallet.     |
| `tokenBalances`         | `TokenBalanceConnection` | Token balances stored in the wallet.             |
| `walletSummary`         | `WalletSummary`          | Summary of assets and tokens in the wallet.      |
| `recent`                | `Boolean`                | Indicates if the wallet has recent transactions. |
| `lastActivityTimestamp` | `Int`                    | Timestamp of the last wallet activity.           |

#### Values

| Name                | Description                        |
| ------------------- | ---------------------------------- |
