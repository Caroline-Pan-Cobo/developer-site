本指南解释了如何检查和管理您的 Cobo CLI 版本。您将学习如何验证当前运行的版本以及如何更新到最新版本以访问新功能和改进。

## 检查当前版本

要显示 Cobo CLI 的当前版本，请使用以下命令：

```shell
cobo version
```

示例输出：

```shell
Cobo CLI version: 0.1.0
```

## 更新 Cobo CLI

为了确保您可以访问最新功能和错误修复，保持 Cobo CLI 更新是很重要的。更新过程取决于您最初如何安装该工具。

如果您使用 pip 安装了 Cobo CLI，可以使用以下命令进行更新：

```shell
pip install --upgrade cobo-cli
```

如果您在 macOS 上使用 Homebrew 安装了 Cobo CLI，可以使用以下命令进行更新：

```shell
brew upgrade cobo-cli
```

更新后，验证新版本：

```shell
cobo version
```
