使用此命令获取并更新最新的 WaaS 2.0 OpenAPI 规范。Cobo CLI 下载 YAML 文件并将其保存到您的本地配置目录，确保您拥有最新的 API 定义。

## 用法

```shell
cobo doc <topic_or_path> [options]
```

## 参数

`<topic_or_path>`：指定文档主题或 API 操作路径。可能的值包括：
   - `guides`：（默认）打开 [WaaS 2.0 简介](/v2_cn/guides/overview/introduction) 指南。
   - `api`：打开 [API 参考](/v2/api-references/playground) 页面。
   - `sdk`：打开 [WaaS SDKs](/v2_cn/developer-tools/quickstart-python) 指南。
   - `apps`：打开 [Cobo Portal Apps开发](/v2_cn/apps/introduction) 指南。
   - API 端点路径：显示指定路径的 API 操作的详细信息，例如 `/wallets`。

## 选项

`-u`, `--update`：更新 WaaS 2.0 的 OpenAPI 规范，确保您拥有最新的 API 定义和文档。使用此选项时，此命令将仅执行更新然后退出。成功更新后，Cobo CLI 将确认操作并显示“规范文件下载成功”消息。

## 示例

```shell
cobo doc /wallets/mpc/vaults
```

此命令获取 `/wallets/mpc/vaults` 端点的文档，包括其描述、参数和响应格式。
