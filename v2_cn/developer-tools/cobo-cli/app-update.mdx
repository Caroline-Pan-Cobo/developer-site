使用 `app update` 命令在 Cobo Portal 上部署 Cobo Portal Apps 的更新。有关更新 Cobo Portal Apps 的更多详细信息，请参阅 [更新 App ](/v2_cn/apps/build-app#update-the-app)。

在更新您的 Cobo Portal Apps 之前，请确保以下事项：
- 验证 [应用Manifest 文件](/v2_cn/apps/build-app#configure-the-manifest-file) 配置正确。
- 使用您的 Cobo Portal 账户登录 Cobo CLI。
- 从包含Manifest 文件的项目目录中运行命令。

## 用法
```shell
cobo app update
```

## 示例
```shell
cobo app update
```
此命令使用最新更改更新 Cobo Portal Apps 。
