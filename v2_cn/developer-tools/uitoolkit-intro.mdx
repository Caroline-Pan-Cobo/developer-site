Cobo UI Toolkit 是一个主要用于促进 [Cobo Portal Apps ](/v2_cn/apps/introduction)前端与 [Cobo Portal](https://manuals.cobo.com/en/portal/introduction) 之间通信的库。这个Cobo UI Toolkit提供了一系列实用工具，可以帮助您减少编码工作量，简化开发过程，并在 App 之间保持用户体验的一致性。

## 主要特性

- **实用函数**：提供一系列实用函数，简化常见任务，如多因素身份验证（MFA）处理和 App 用户验证，提高开发效率并减少样板代码。
- **框架集成**：专为与 React 无缝集成而设计，便于整合到现有项目中。

## 安装

要在您的项目中使用 Cobo UI Toolkit，您首先需要通过 npm 安装它。您可以在终端窗口中运行以下命令：

```
npm install @cobo/cobo-ui-toolkit
```
