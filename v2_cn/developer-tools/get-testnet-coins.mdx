要测试您与 WaaS 2.0 API 的集成，您可以使用 Sepolia（ETH 测试网）和比特币测试网环境。本指南说明如何从可靠的来源获取测试网代币。

## Sepolia 测试网 (SETH)

使用 Sepolia 测试网测试基于以太坊的交易。使用以下工具监控您的交易：

- **区块浏览器**：[Etherscan](https://sepolia.etherscan.io/)

### 从水龙头获取 SETH

从这些可靠的水龙头服务获取 SETH：

- [QuickNode Sepolia 水龙头](https://faucet.quicknode.com/drip)
- [Alchemy Sepolia 水龙头](https://sepoliafaucet.com/)
- [Grabteeth](https://grabteeth.xyz/)
- [PoW 水龙头](https://sepolia-faucet.pk910.de/)

## 比特币测试网 (XTN)

使用比特币测试网测试 BTC 交易。使用以下工具监控您的交易：

- **区块浏览器**：[BlockCypher](https://live.blockcypher.com/btc-testnet/)

### 从水龙头获取 XTN

从这些可靠的水龙头服务获取 XTN：

- [Bitaps 比特币测试网水龙头](https://tbtc.bitaps.com/)
- [比特币测试网水龙头](https://bitcoinfaucet.uo1.net/)
- [Coinfaucet](https://coinfaucet.eu/en/btc-testnet/)
- [Cryptopump](https://cryptopump.info/)

## 最佳实践

- 将测试网地址与生产地址分开
- 仅请求测试所需的数量
- 在可能的情况下将未使用的测试网代币返还给水龙头
