当 TSS Node 包的新版本发布时，强烈建议通过升级到最新版本来保持更新。在下载 TSS Node 包的新版本后，请确保完成以下步骤。

1. 停止当前运行的 TSS Node。

```
sudo ./tss-node.sh stop
```

2. 备份关键数据（例如 `secrets.db` 文件、密码或 `password.key` 文件以及配置文件）。

3. 解压缩 TSS Node 包的新版本并覆盖旧版本中的文件。您可以在以前 TSS Node 包使用的同一文件夹中解压缩新版本。

```
tar -xzf cobo-tss-node-generic.tgz
```

4. 为确保 TSS Node 的正常运行，请检查其操作环境并拉取必要的容器镜像。

```
sudo ./tss-node.sh status
```

5. 启动新版本的 TSS Node ，并通过确保日志中的版本号和关键数据（如 TSS Node  ID）与预期值匹配来验证其成功启动。

```
sudo ./tss-node.sh start
```
