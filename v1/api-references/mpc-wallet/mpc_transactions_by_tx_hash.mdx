---
title: Get Transactions By Tx Hash
api: GET /v1/custody/mpc/transactions_by_tx_hash/ 
description: <Note>This content applies to WaaS 1.0 only. We highly recommend that you upgrade to <a href="/v2/guides/overview/introduction">WaaS 2.0</a>.</Note> This endpoint returns transaction details for a given transaction hash. The response is a JSON object containing transaction details such as transaction hash, block number, sender address, recipient address, value, and gas used. The endpoint also accepts an optional parameter 'transaction_type' to filter the transaction details based on the type of transaction.
---
