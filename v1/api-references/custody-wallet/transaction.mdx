---
title: Get Transaction Details
api: "GET /v1/custody/transaction/"
description: <Note>This content applies to WaaS 1.0 only. We highly recommend that you upgrade to [WaaS 2.0](/v2/guides/overview/introduction).</Note> This endpoint retrieves a transaction by its ID from the WaaS API. The endpoint requires authentication and takes a single query parameter, 'id', which is the ID of the transaction to retrieve.
---
<Note>Only successful deposit/withdraw requests will be considered as 'Transaction' here</Note>

<ParamField query="id" type="String" required>Cobo Unique Transaction ID</ParamField>
